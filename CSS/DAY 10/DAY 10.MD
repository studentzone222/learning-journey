📚 Today's Topic: CSS Fundamentals

Today I learned several important CSS concepts that help control the layout and appearance of web pages.

1. Inline, Internal, and External CSS

I learned the three ways to apply CSS to HTML:

Inline CSS – Styles are written directly inside an HTML element using the style attribute.
Internal CSS – Styles are written inside a <style> tag in the HTML document.
External CSS – Styles are stored in a separate .css file and linked to the HTML using the <link> tag. This is the best approach for larger projects because it keeps the code organized and reusable.
2. Width and Height

I learned how to control the size of elements using:

width
height
min-width
max-width
min-height
max-height

These properties help create responsive layouts and prevent elements from becoming too large or too small.

3. CSS Combinators

I learned how combinators target elements based on their relationships:

Descendant ( )
Child (>)
Adjacent sibling (+)
General sibling (~)

These selectors make it easier to style elements without adding extra classes or IDs.

4. Inline vs Block-Level Elements

I learned the difference between:

Inline elements – Only take up the space they need and stay on the same line (for example, span).
Block-level elements – Take up the full available width and start on a new line (for example, div, section, and p).
5. Inline-Block

I learned that display: inline-block combines features of both inline and block elements. It allows elements to stay on the same line while also accepting width, height, margin, and padding.

6. Margin and Padding

I learned the difference between:

Margin – Creates space outside an element.
Padding – Creates space inside an element, between the content and the border.

Understanding the CSS Box Model is essential for building clean and well-spaced layouts.
